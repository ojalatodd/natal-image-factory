# Lode Coding Session

Follow these steps to start a Lode Coding session for the Natal Image Factory:

1. Read the system prompt at `prompts/SystemPrompt.txt` and follow its instructions for the duration of this session.

2. Read these three files to seed your context:
   - `lode/lode-map.md` — index of all lode documentation
   - `lode/terminology.md` — domain language
   - `lode/summary.md` — project snapshot

3. Briefly show that you have domain knowledge by summarizing the project in 2-3 sentences.

4. Ask the user what they'd like to work on.

5. Before exploring the codebase or searching for files, ALWAYS check `lode/lode-map.md` first to find relevant lode files.

6. After completing any code change, update the corresponding lode file(s) to reflect the new current state of the system.
